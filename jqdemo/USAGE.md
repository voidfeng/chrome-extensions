# 使用指南

## 🎯 快速开始

### 1️⃣ 打开页面
在浏览器中打开 `index.html` 文件

### 2️⃣ 上传字典
- **方式一**：点击"选择 MDX 文件"按钮
- **方式二**：直接拖拽 `.mdx` 文件到上传区域

### 3️⃣ 浏览词条
上传成功后，页面会自动显示 **10个随机词条**

```
📖 随机词条（点击查看释义）
┌─────────┬─────────┬─────────┐
│ example │ hello   │ world   │
├─────────┼─────────┼─────────┤
│ apple   │ banana  │ orange  │
├─────────┼─────────┼─────────┤
│ book    │ table   │ chair   │
└─────────┴─────────┴─────────┘
        🔄 换一批
```

### 4️⃣ 查看释义
- **点击词条**：直接点击任意词条查看释义
- **手动输入**：在搜索框输入单词并按回车

### 5️⃣ 返回列表
查看完释义后，点击 **"← 返回词条列表"** 按钮

## 💡 使用技巧

### 快速浏览
1. 上传字典后，先浏览随机词条
2. 点击感兴趣的词条查看详情
3. 点击"换一批"发现更多词条

### 精确查询
1. 在搜索框输入完整单词
2. 按回车键或点击"查询"按钮
3. 查看详细释义

### 批量学习
1. 多次点击"换一批"按钮
2. 每次显示10个新的随机词条
3. 适合随机学习和复习

## ❓ 常见问题

### Q: 为什么只显示10个词条？
A: 为了提供更好的浏览体验，每次显示10个。可以点击"换一批"查看更多。

### Q: 词条是完全随机的吗？
A: 是的，每次点击"换一批"都会从字典中随机选择10个不同的词条。

### Q: 可以调整显示数量吗？
A: 可以，在代码中修改 `getRandomWords(keys, 10)` 的数字即可。

### Q: 如何快速学习单词？
A: 建议使用"换一批"功能，每次学习10个新词条，循环往复。

## 🎓 学习建议

3. **重复复习**：多次点击"换一批"，加深印象

## 🚀 高级用法

### 自定义显示数量
编辑 `index.html`，找到：
```javascript
const randomKeys = getRandomWords(keys, 10);
```
将 `10` 改为你想要的数量。

### 修改样式
编辑 `<style>` 标签中的 CSS：
```css
.word-item {
    /* 自定义词条样式 */
}
```

### 添加收藏功能
可以扩展代码，添加收藏词条的功能。
