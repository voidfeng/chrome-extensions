# 快速开始

## 🚀 三步启动

### 第一步：启动服务器

**Mac/Linux 用户：**
```bash
./start-server.sh
```

**Windows 用户：**
双击运行 `start-server.bat`

**或者手动启动（Python）：**
```bash
python3 -m http.server 8000
```

### 第二步：打开浏览器

在浏览器中访问：
```
http://localhost:8000
```

### 第三步：上传字典

1. 点击"选择 MDX 文件"按钮
2. 选择你的 `.mdx` 字典文件
3. 等待加载完成
4. 开始查询单词！

---

## ⚠️ 重要提示

**不能直接双击打开 `index.html` 文件！**

由于使用了 ES Module，必须通过本地服务器访问，否则会遇到 CORS 错误。

---

## 💡 常见问题

### 没有 Python？

**安装 Python：**
- Mac: `brew install python3`
- Windows: 从 [python.org](https://www.python.org/downloads/) 下载安装

**或使用 Node.js：**
```bash
npx http-server -p 8000
```

### 端口被占用？

修改端口号：
```bash
python3 -m http.server 8080  # 使用 8080 端口
```

然后访问 `http://localhost:8080`

---

## 🎉 开始使用

服务器启动后，你会看到：
```
✅ 服务器已启动！
📱 请在浏览器中打开: http://localhost:8000
```

### 测试库是否正常加载

先访问测试页面：
```
http://localhost:8000/test.html
```

如果看到"所有库加载成功！"，说明环境正常。

然后访问主页面：
```
http://localhost:8000/index.html
```

现在就可以开始使用 MDX 字典查询工具了！
